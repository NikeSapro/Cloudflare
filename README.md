# Představení platformy Cloudflare

## Odkazy
* **Živá stránka (GitHub Pages):** https://nikesapro.github.io/Cloudflare/

## Obsah projektu
* `index.html` - Struktura webové stránky s textovým obsahem.
* `style.css` - Vlastní stylování stránky v barvách Cloudflare (oranžová, tmavě modrá).
* `README.md` - Dokumentace projektu.

## Funkcionalita a splněné požadavky
* **Co je Cloudflare:** Stručné vysvětlení v rozsahu 3 vět.
* **K čemu slouží:** Popis přínosu platformy vlastními slovy.
* **Vybrané služby:** Detailní představení 3 klíčových služeb (CDN, DDoS ochrana, Správa DNS), přičemž každá obsahuje jednoúčelový popis v jedné větě.
* **Responzivní design:** Stránka se správně zobrazuje na mobilech, tabletech i počítačích díky CSS Gridu.

## Jak projekt spustit lokálně
1. Stáhněte nebo klonujte tento repozitář.
2. Otevřete soubor `index.html` v jakémkoliv webovém prohlížeči (Chrome, Firefox, Safari, Edge).
